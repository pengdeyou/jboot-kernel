# JBoot-Kernel 4.0 使用手册

欢迎使用 JBoot-Kernel 4.0 使用手册！

JBoot-Kernel 是一个基于 Spring Boot 3.x 和 Java 17 构建的强大工具集，为 JBoot 框架提供核心功能支持。

## 特性

- ✅ 基于 Spring Boot 3.x 和 Java 17
- ✅ 模块化设计，易于扩展和使用
- ✅ 内置丰富的工具类和组件
- ✅ 支持微服务架构
- ✅ 提供完整的权限管理和安全功能
- ✅ 内置多数据源支持
- ✅ 丰富的开发工具和代码生成器

## 快速开始

### 环境要求

- JDK 17+
- Maven 3.6+
- Spring Boot 3.x

### 安装

在 Maven 项目中添加以下依赖：

```xml
<dependency>
    <groupId>org.jboot</groupId>
    <artifactId>jboot-kernel</artifactId>
    <version>4.7.1</version>
</dependency>
```

### 模块文档

请查看 [模块文档](/modules/) 了解各个模块的详细使用说明。
